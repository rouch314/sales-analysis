# Анализ продаж супермаркета (Kaggle dataset)

[Ссылка на датасет](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting)

![Dataset cover](https://storage.googleapis.com/kaggle-datasets-images/835308/1426440/aaf8255ae75bbbec8c79afb01c5f3314/dataset-cover.jpg?t=2020-09-11-15-39-15)

## ЦЕЛЬ:
Проанилизировать статистику и структуру продаж для выявления ключевых драйверов генерации и роста выручки.

### Ключевые выводы:
- Самая продаваемая категория товаров - *"Техника"*.
- Пик продаж приходится на 4 квартал каждого года.
- Штаты с самыми большими продажами: **Калифорния** и **Нью-Йорк**.
- Есть ряд штатов, где наблюдается волатильность показателя продаж.
- Наблюдается правосторонняя ассиметрия распределения значений продаж - б*о*льшую часть выручки генерирует малая часть дорогих товаров.
- Невозможно расчитать прибыль и рентабельность продаж, что обедняет результаты анализа.

### Рекомендации:
1. Масштабировать ассортимент в категории "Техника" и усилить маркетинг - в особенности, ближе к 4 кварталу года.
2. Проанализировать продажи в штатах **Мичиган, Пенсильвания, Техас, Вашингтон, Вирджиниа, Иллинойс, Флорида**. Есть потенциал роста продаж в этих штатах при реализации правильной стратегии.
